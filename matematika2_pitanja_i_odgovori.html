I — Primitivna funkcija i neodređeni integral
klikni
P1
Definisati primitivnu funkciju. Kada kažemo da je F(x) primitivna funkcija za f(x) na (a,b)?
✓ Odgovor
F = F(x) je tačna primitivna funkcija za f = f(x) na (a,b) ako je:
  • F ∈ D((a,b)) — F je diferencijabilna na (a,b)
  • važi F'(x) = f(x) za sve x ∈ (a,b)
Primjer: F(x) = x² je primitivna za f(x) = 2x, jer (x²)' = 2x. Takođe x²+1, x²+5, ... sve su primitivne za f(x) = 2x.
klikni
P2
Formulisati i dokazati teoremu o obliku skupa svih primitivnih funkcija za f na (a,b).
✓ Odgovor
Teorema: Ako je F tačna primitivna funkcija za f, tada su sve tačne primitivne funkcije oblika F + c, c ∈ ℝ.

Dokaz:
Neka je G još jedna primitivna funkcija za f, tj. G' = f.
Tada: (G − F)' = G' − F' = f − f = 0
Neka H = G − F. H je takva da H' = 0 na (a,b).
Uzmimo x₁, x₂ ∈ (a,b). Po Lagrangeovoj teoremi:
H(x₁) − H(x₂) = H'(s) · (x₁ − x₂), s ∈ (x₁, x₂)
Pošto H'(s) = 0, slijedi H(x₁) − H(x₂) = 0, tj. H(x₁) = H(x₂).
H uzima istu vrijednost u svim tačkama ⟹ H je konstanta, H(x) = c, c ∈ ℝ.
Dakle G − F = c ⟹ G = F + c. □
klikni
P3
Šta je neodređeni integral? Navesti oznaku i objasniti šta svaki dio oznake predstavlja.
✓ Odgovor
Neodređeni integral predstavlja skup svih tačnih primitivnih funkcija za f.
∫ f(x) dx = F(x) + C
  • ∫ — znak integrala
  • f(x) — podintegralna funkcija (ono što integrišemo)
  • dx — podintegralni izraz / mjera integracije
  • F(x) — jedna primitivna funkcija za f
  • C — konstanta integracije, C ∈ ℝ
klikni
P4
Navesti sva 4 osnovna svojstva neodređenog integrala.
✓ Odgovor
  1. Derivacija integrala:
    (∫ f(x) dx)' = f(x)
  2. Diferencijal integrala:
    d(∫ f(x) dx) = (∫ f(x) dx)' dx = f(x) dx
  3. Integral derivacije:
    ∫ f'(x) dx = f(x) + C, C ∈ ℝ
  4. Linearnost (zbir i skaliranje):
    ∫ [αf(x) + βg(x)] dx = α∫f(x)dx + β∫g(x)dx, α,β ∈ ℝ
II — Tabela integrala
klikni
P5
Napisati svih 10 formula iz tabele neodređenih integrala.
✓ Odgovor
1) ∫ xᵐ dx = x^(m+1)/(m+1) + C, m ∈ ℕ 2) ∫ xᵅ dx = x^(α+1)/(α+1) + C, α ∈ ℝ\{−1} 3) ∫ (1/x) dx = ln|x| + C, x ≠ 0 4) ∫ sin(x) dx = −cos(x) + C 5) ∫ cos(x) dx = sin(x) + C 6) ∫ 1/cos²(x) dx = tg(x) + C, x ≠ π/2 + kπ 7) ∫ 1/sin²(x) dx = −ctg(x) + C, x ≠ kπ 8) ∫ aˣ dx = aˣ/ln(a) + C, a > 0, a ≠ 1 (Specijalno: ∫ eˣ dx = eˣ + C) 9) ∫ 1/(1+x²) dx = arctg(x) + C ili −arcctg(x) + C 10) ∫ 1/√(1−x²) dx = arcsin(x) + C ili −arccos(x) + C, −1 < x < 1
Bonus (iz primjera): ∫ 1/√(x²±a²) dx = ln|x + √(x²±a²)| + C, a ≠ 0

Izračunati direktno iz tabele (linearnost + tabela):
∫ [∛x + 4/x − sin(x)] dx
✓ Odgovor
∫ [x^(1/3) + 4/x − sin(x)] dx = ∫ x^(1/3) dx + 4·∫ (1/x) dx − ∫ sin(x) dx = x^(4/3)/(4/3) + 4·ln|x| − (−cos(x)) + C
= (3/4)·x^(4/3) + 4·ln|x| + cos(x) + C

P8
Koristiti svojstvo 3 (integral derivacije) i izračunati:
∫ 2(x⁴+3x)·(4x³+3) dx
✓ Odgovor

Integrand je oblika 2·φ(x)·φ'(x) = (φ²)'
∫ 2(x⁴+3x)(4x³+3) dx = (x⁴+3x)² + C
III — Supstitucija (smjena promjenljive)
klikni
P9
Formulisati teoremu o smjeni promjenljive (supstituciji) u neodređenom integralu.
✓ Odgovor
Teorema: Neka je F tačna primitivna funkcija za f na (α,β). Neka je na (a,b) φ = φ(x) diferencijabilna i φ(a,b) ⊆ (α,β). Tada:
∫ f(φ(x)) · φ'(x) dx = F(φ(x)) + C
Postupak supstitucije:
  1. Postaviti smjenu: t = φ(x)
  2. Izračunati: dt = φ'(x) dx
  3. Izraziti dx: dx = dt/φ'(x)
  4. Supstituisati, izračunati integral po t
  5. Vratiti originalnu promjenljivu: t → φ(x)
klikni
P10
Izračunati supstitucijom:
∫ e^(4x+2) dx
✓ Odgovor
Supstitucija: t = 4x+2, dt = 4dx ⟹ dx = dt/4 ∫ eᵗ · (dt/4) = (1/4) ∫ eᵗ dt = (1/4)eᵗ + C
= (1/4)·e^(4x+2) + C
klikni
P11
Izračunati supstitucijom:
∫ (2x+5)⁴ dx
✓ Odgovor
t = 2x+5, dt = 2dx ⟹ dx = dt/2 ∫ t⁴ · (dt/2) = (1/2) · t⁵/5 + C
= (1/10)·(2x+5)⁵ + C
klikni
P12
Izračunati supstitucijom:
∫ e^(x²) · x dx
✓ Odgovor
t = x², dt = 2x dx ⟹ x dx = dt/2 ∫ eᵗ · (dt/2) = (1/2) eᵗ + C
= (1/2)·e^(x²) + C
klikni
P13
Izračunati supstitucijom:
∫ 1/(x² + a²) dx, a ≠ 0
✓ Odgovor
∫ 1/(x²+a²) dx = (1/a²) ∫ 1/(1+(x/a)²) dx Supstitucija: t = x/a, dt = dx/a ⟹ dx = a·dt = (1/a²) · a · ∫ 1/(1+t²) dt = (1/a) arctg(t) + C
= (1/a)·arctg(x/a) + C
klikni
P14
Izračunati supstitucijom (trigonometrijska supstitucija):
∫ √(a²−x²) dx, a > 0
✓ Odgovor
Supstitucija: x = a·sin(t), dx = a·cos(t) dt, t = arcsin(x/a) Uslov: −π/2 < t < π/2 (cos(t) ≥ 0) √(a²−x²) = √(a²−a²sin²t) = a·√(1−sin²t) = a·|cos(t)| = a·cos(t) ∫ a·cos(t) · a·cos(t) dt = a² ∫ cos²(t) dt cos²(t) = (1 + cos(2t))/2 = a² ∫ (1/2 + cos(2t)/2) dt = a²(t/2 + sin(2t)/4) + C = (a²/2)(t + sin(2t)/2) + C = (a²/2)(arcsin(x/a) + (1/2)·sin(2·arcsin(x/a))) + C
= (a²/2)·arcsin(x/a) + (x/2)·√(a²−x²) + C
klikni
P15
Izračunati:
∫ 1/√(x²±a²) dx, a ≠ 0
✓ Odgovor
Supstitucija: t = x + √(x²±a²)
dt = (1 + x/√(x²±a²)) dx = (√(x²±a²) + x)/√(x²±a²) · dx = t/√(x²±a²) · dx ⟹ dx/√(x²±a²) = dt/t ∫ dt/t = ln|t| + C
= ln|x + √(x²±a²)| + C
klikni
P16
Izračunati:
∫ 1/(x² + 4x + 13) dx
✓ Odgovor
Diskriminanta: D = 16 − 52 < 0 ⟹ dopuniti kvadrat x²+4x+13 = x²+2·x·2+4+9 = (x+2)²+9 = (x+2)²+3² ∫ 1/((x+2)²+9) dx = (1/9) ∫ 1/(1+((x+2)/3)²) dx Supstitucija: t = (x+2)/3, dx = 3·dt = (1/9)·3 ∫ 1/(1+t²) dt = (1/3) arctg(t) + C
= (1/3)·arctg((x+2)/3) + C
klikni
P17
Izračunati:
∫ 1/(x² − 4x + 4) dx
✓ Odgovor
D = 16−16 = 0 ⟹ x₁,₂ = 2 (dvostruki nul) x²−4x+4 = (x−2)² ∫ 1/(x−2)² dx = ∫ (x−2)^(−2) dx Supstitucija: t = x−2, dt = dx = ∫ t^(−2) dt = t^(−1)/(−1) + C = −1/t + C
= −1/(x−2) + C
IV — Parcijalna integracija
klikni
P18
Formulisati teoremu o parcijalnoj integraciji. Navesti oba zapisa formule.
✓ Odgovor
Teorema: Neka su u, v ∈ D(a,b) i neka postoji ∫u'(x)v(x)dx. Tada postoji i ∫u(x)v'(x)dx i važi:
∫ u(x)·v'(x) dx = u(x)·v(x) − ∫ u'(x)·v(x) dx
Drugi zapis (diferencijalni oblik):
∫ u dv = u·v − ∫ v du
Ideja (zašto vrijedi):
(u·v)' = u'·v + u·v' ⟹ u·v' = (u·v)' − u'·v
Integrišemo obje strane: ∫u·v'dx = u·v − ∫u'·v dx
klikni
P19
Izračunati parcijalnom integracijom:
∫ x·eˣ dx
✓ Odgovor
u = x du = dx dv = eˣdx v = eˣ ∫ x·eˣ dx = x·eˣ − ∫ eˣ dx = x·eˣ − eˣ + C
= eˣ(x − 1) + C
Provjera: (eˣ(x−1))' = eˣ(x−1) + eˣ·1 = eˣ·x ✓
klikni
P20
Izračunati parcijalnom integracijom (dva puta):
∫ x²·sin(x) dx
✓ Odgovor
Trik "tablični metod" (tablica deriviranja/integriranja):
Deriviramo: x² → 2x → 2 → 0
Integrišemo: sin(x) → −cos(x) → −sin(x) → cos(x)
Naizmjenični znakovi: +, −, +
= (+1)·x²·(−cos(x)) + (−1)·2x·(−sin(x)) + (+1)·2·cos(x) + C
= −x²·cos(x) + 2x·sin(x) + 2·cos(x) + C
klikni
P21
Izračunati parcijalnom integracijom (vraćanje integrala):
∫ eˣ·cos(x) dx
✓ Odgovor
Označimo I = ∫ eˣ·cos(x) dx 1. korak: u = eˣ, dv = cos(x)dx ⟹ du = eˣdx, v = sin(x) I = eˣ·sin(x) − ∫ eˣ·sin(x) dx 2. korak: u = eˣ, dv = sin(x)dx ⟹ du = eˣdx, v = −cos(x) ∫ eˣ·sin(x)dx = eˣ·(−cos(x)) − ∫ eˣ·(−cos(x))dx = −eˣ·cos(x) + ∫ eˣ·cos(x) dx = −eˣ·cos(x) + I Dakle: I = eˣ·sin(x) − (−eˣ·cos(x) + I) I = eˣ·sin(x) + eˣ·cos(x) − I 2I = eˣ·sin(x) + eˣ·cos(x) + C
I = (1/2)·eˣ·(sin(x) + cos(x)) + C
klikni
P22
Izračunati parcijalnom integracijom:
∫ arctg(x) dx
✓ Odgovor
u = arctg(x) du = 1/(1+x²) dx dv = dx v = x = x·arctg(x) − ∫ x/(1+x²) dx Preostali integral: supstitucija t = 1+x², dt = 2x dx ∫ x/(1+x²) dx = (1/2) ∫ dt/t = (1/2)·ln|t| = (1/2)·ln(1+x²)
= x·arctg(x) − (1/2)·ln(1+x²) + C
klikni
P23
Izračunati parcijalnom integracijom (tablični metod):
∫ x³·cos(x) dx
✓ Odgovor
Tablični metod: Deriviramo (+/−): x³ → 3x² → 6x → 6 → 0 Integrišemo: cos(x) → sin(x) → −cos(x) → −sin(x) → cos(x) Znakovi: + − + − = x³·sin(x) + 3x²·cos(x) − 6x·sin(x) − 6·cos(x) + C
= x³·sin(x) + 3x²·cos(x) − 6x·sin(x) − 6·cos(x) + C
klikni
P24
Izračunati parcijalnom integracijom:
∫ √(a²+x²) dx
✓ Odgovor
u = √(a²+x²) du = x/√(a²+x²) dx dv = dx v = x I = x√(a²+x²) − ∫ x²/√(a²+x²) dx Dodajemo i oduzimamo a²: = x√(a²+x²) − ∫ (x²+a²−a²)/√(a²+x²) dx = x√(a²+x²) − ∫ √(a²+x²) dx + a² ∫ 1/√(a²+x²) dx = x√(a²+x²) − I + a²·ln|x+√(a²+x²)| + C 2I = x√(a²+x²) + a²·ln|x+√(a²+x²)| + C
= (1/2)(x√(a²+x²) + a²·ln|x+√(a²+x²)|) + C
V — Integracija racionalnih funkcija
klikni
P25
Opisati opštu metodu integracije racionalnih funkcija Pₘ(x)/Qₙ(x). Koji su koraci?
✓ Odgovor
Korak 1: Ako je m ≥ n, dijeljenje polinoma:
Pₘ(x) = S(x)·Qₙ(x) + r(x), deg(r) < deg(Q) ∫ Pₘ/Qₙ dx = ∫ S(x) dx + ∫ r(x)/Qₙ(x) dx
Korak 2: Faktorizacija Qₙ(x) (nalazimo nule):
  • Realne nule → linearni faktori (x−aᵢ)^kᵢ
  • Kompleksne nule u parovima → kvadratni faktori (x²+pⱼx+qⱼ)^lⱼ gdje D = p²−4q < 0
Korak 3: Razlaganje na parcijalne razlomke (4 oblika):
(i) A/(x−a) (ii) A/(x−a)^k, k ≥ 2 (iii) (Ax+B)/(x²+px+q), D < 0 (iv) (Ax+B)/(x²+px+q)^k, D < 0, k ≥ 2
Korak 4: Određivanje neodređenih koeficijenata (A, B, C, D, ...).
Korak 5: Integracija svakog razlomka posebno.
klikni
P26
Navesti formule za integraciju sva 4 osnovna oblika parcijalnih razlomaka.
✓ Odgovor
(i) ∫ A/(x−a) dx = A·ln|x−a| + C (ii) ∫ A/(x−a)^k dx = A·(x−a)^(1−k)/(1−k) + C, k ≥ 2 (iii) ∫ (Ax+B)/(x²+px+q) dx (D = p²−4q < 0): = (A/2)·ln|x²+px+q| + (B−Ap/2)·(1/a)·arctg((x+p/2)/a) + C gdje je a² = q − p²/4 > 0 (iv) ∫ (Ax+B)/(x²+px+q)^k dx → svodi se na Iₖ rekurentnom formulom: Iₖ₊₁ = 1/(2ka²) · [t/(t²+a²)^k + (2k−1)·Iₖ] gdje t = x + p/2, a² = q − p²/4
klikni
P27
Izračunati integracijom racionalne funkcije:
∫ 1/(x² − 2x − 3) dx
✓ Odgovor
D = 4+12 = 16 > 0 ⟹ x₁ = 3, x₂ = −1 x²−2x−3 = (x−3)(x+1) Razlaganje: 1/((x−3)(x+1)) = A/(x−3) + B/(x+1) 1 = A(x+1) + B(x−3) 1 = (A+B)x + (A−3B) A+B = 0 ⟹ A = −B A−3B = 1 ⟹ −4B = 1 ⟹ B = −1/4, A = 1/4 ∫ (1/4/(x−3) − 1/4/(x+1)) dx = (1/4)·ln|x−3| − (1/4)·ln|x+1| + C
= (1/4)·ln|(x−3)/(x+1)| + C
klikni
P28
Izračunati integracijom racionalne funkcije (nepravi razlomak — dijeljenje polinoma):
∫ (8x³ + 4x² + 5x − 6) / (x² − 9) dx
✓ Odgovor
Stepeni: m=3, n=2 ⟹ m > n, dijelimo: (8x³+4x²+5x−6) : (x²−9) = 8x+4 ostatak 77x+30 ⟹ 8x³+4x²+5x−6 = (x²−9)(8x+4) + 77x+30 ∫ = ∫(8x+4)dx + ∫(77x+30)/(x²−9) dx x²−9 = (x−3)(x+3) (77x+30)/((x−3)(x+3)) = A/(x−3) + B/(x+3) 77x+30 = A(x+3)+B(x−3) x=3: 261 = 6A ⟹ A = 261/6 x=−3: −201 = −6B ⟹ B = 201/6 ∫(8x+4)dx = 4x² + 4x
= 4x²+4x + (261/6)·ln|x−3| + (201/6)·ln|x+3| + C
klikni
P29
Napisati razlaganje na parcijalne razlomke (bez računanja koeficijenata) za:
∫ (4x+2) / (x⁴−x³−x²−x−2) dx
gdje je: x⁴−x³−x²−x−2 = (x−2)(x+1)(x²+1)
✓ Odgovor
(4x+2)/((x−2)(x+1)(x²+1)) = A/(x−2) + B/(x+1) + (Cx+D)/(x²+1)
Odrediti A, B, C, D množenjem sa nazivnikom i poređenjem koeficijenata ili uvrštavanjem nula.
x²+1 = 0 nema realnih rješenja (D = −4 < 0) → kvadratni faktor s linearnim brojevnikom.
VI — Napredni integrali i rekurentne formule
klikni
P30
Izvesti rekurentnu formulu za:
Iₖ = ∫ 1/(t²+a²)^k dt
i primijeniti je za izračunavanje I₃ za slučaj a = 2 (tj. ∫ 1/(x²+4)³ dx).
✓ Odgovor
Izvod rekurentne formule (parcijalnom integracijom):
Iₖ = ∫ 1/(t²+a²)^k dt u = (t²+a²)^(−k) du = −k·(t²+a²)^(−k−1)·2t dt dv = dt v = t = t·(t²+a²)^(−k) + 2k ∫ t²/(t²+a²)^(k+1) dt Dodamo i oduzmemo a²: t² = (t²+a²) − a² = t/(t²+a²)^k + 2k·Iₖ − 2k·a²·Iₖ₊₁ ⟹ Iₖ = t/(t²+a²)^k + 2k·Iₖ − 2k·a²·Iₖ₊₁ ⟹ 2k·a²·Iₖ₊₁ = t/(t²+a²)^k + (2k−1)·Iₖ
Iₖ₊₁ = 1/(2k·a²) · [t/(t²+a²)^k + (2k−1)·Iₖ]
Primjena za I₃ (a=2, k=2):
I₁ = (1/2)·arctg(x/2) + C (iz tabele, a=2) I₂ = 1/(2·1·4) · [x/(x²+4)¹ + 1·I₁] = (1/8) · [x/(x²+4) + (1/2)·arctg(x/2)] + C I₃ = 1/(2·2·4) · [x/(x²+4)² + 3·I₂] = (1/16) · [x/(x²+4)² + 3·(1/8)·(x/(x²+4) + (1/2)·arctg(x/2))] + C
klikni
B1
(Bonus — integracioni put) Kada se za integral oblika ∫ 1/(ax²+bx+c) dx koristi arctg, a kada parcijalni razlomci? Objasniti kriterij.
✓ Odgovor
Kriterij je diskriminanta D = b² − 4ac:
D < 0: kvadratni trinomial nema realnih nula ⟹ dopunjujemo kvadrat → oblik (x+p/2)²+a² ⟹ koristimo arctg formulu D = 0: dvostruki nul x₀ = −b/2a ⟹ ax²+bx+c = a(x−x₀)² ⟹ ∫ 1/((x−x₀)²) dx = −1/(x−x₀) + C D > 0: dva realna nula x₁, x₂ ⟹ ax²+bx+c = a(x−x₁)(x−x₂) ⟹ parcijalni razlomci: A/(x−x₁) + B/(x−x₂) ⟹ rezultat ln formulama
klikni
B2
(Bonus) Izračunati:
∫ (Ax+B)/(x²+px+q) dx, gdje je D = p²−4q < 0
Objasniti opšti postupak.
✓ Odgovor
Ideja: rastaviti brojevnik da se u njemu pojavi (2x+p) = derivacija nazivnika.
Ax+B = (A/2)·(2x+p) + (B − Ap/2) ∫ (Ax+B)/(x²+px+q) dx = (A/2) ∫ (2x+p)/(x²+px+q) dx + (B−Ap/2) ∫ 1/(x²+px+q) dx 1. integral: t = x²+px+q, dt = (2x+p)dx ⟹ (A/2) ∫ dt/t = (A/2)·ln|x²+px+q| 2. integral: dopuniti kvadrat x²+px+q = (x+p/2)² + (q−p²/4) = (x+p/2)² + a² gdje a² = q−p²/4 > 0 (jer D < 0) ⟹ (B−Ap/2)·(1/a)·arctg((x+p/2)/a)
= (A/2)·ln|x²+px+q| + (B−Ap/2)·(1/a)·arctg((x+p/2)/a) + C
